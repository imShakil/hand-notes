
# Dropdown in markdown

## Keep Secret by default

<details>
<summary> Want to disclose hidden things? </summary>
<br>

This is your hidden text.

code:
```
<details>
<summary> Want to disclose hidden things? </summary>
<br>
This is your hidden text.

</details>
```
</details>

## Disclosed by default

<details open>
<summary>Want to hide?</summary>
<br>
This is open by default.

Code: 

```
<details open>
<summary> Want to hide? </summary>
<br>

This is open by default.
</details>
```
</details>
